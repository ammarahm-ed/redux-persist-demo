## Getting Started
**1. Install Dependencies:**
```
npm install
```

**2. Install Cocoapods:**
```
cd ios
pod install
```

**3. Start iOS Simulator:**
```
react-native run-ios
```
**4. Start Android:**
```
react-native run-android
```
